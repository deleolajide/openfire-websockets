WebSockets Plugin Readme

Prerequesites

Installation

  1. Stop Openfire.
  2. Copy the websockets.war file to the OPENFIRE_HOME/plugins directory.
  3. Restart Openfire.
  4. From a browser, go to http://your_openfire-server:7070/websockets/sparkweb
  5. From a browser, go to http://your_openfire-server:7070/websockets/peek
  6. From a browser, go to http://your_openfire-server:7070/websockets/ofchat-010.crx
  7. If these pages do not appear, please check you log files and post any errors on www.igniterealtime.org

Configuration

  • Go to the admin properties page
